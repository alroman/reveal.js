Intro to react tooling

React tooling

React's JSX needs to be transpiled to JS your browser can read. Establishing a transpiler workflow is done similarly to most Javascript build chains. You can use Grun, Gulp, Webpack, or a custom shell command.

Webpack

Webpack is a module bundler in the spirit of Grunt and Gulp. It converts dependencies to static web assets, can split your code into modules, provides lazy loading, runs dev server... and a million other things.

The default installation of React uses Webpack, but it hides a lot of details from you. This is nice to get started, but we're CS people, we need to know what's going on under the hood.
