Intro to react tooling

React tooling

React's JSX needs to be transpiled to JS your browser can read. Establishing a transpiler workflow is done similarly to everythign else. You can use Grun, Gulp, Webpack, or your shell.

Webpack

By default react uses Webpack, but it hides a lot of details from you.

Webpack is a module bundler. Converts dependencies to static web assets, can split your code into modules, provides lazy loading, runs dev server... and a million other things.

https://webpack.js.org
