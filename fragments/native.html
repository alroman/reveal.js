Going native

iOS + Android

React Native takes all the concepts from React.js (as well as the core) and lets you produce fully native apps. No web wrapper, no BS. Fully native apps running at ~ 60fps.

!Cordova

This is not at all like Cordova (or Ionic) or any other web-wrapper platform. The code produced is fully native `Objective-C` and `Java` code.

React Native

How this is done is a cool CS topic onto itself. You can imagine this process as taking the DOM and replacing it with UIKit (or Android equivalent). So instead of rendering HTML nodes, you render UIKit `UIView`s or Android `Activity`s

React Native Sample


import React, { Component } from 'react';

import {
    StyleSheet,
    Text,
    View,
    Button,
    Image,
    TouchableOpacity
} from 'react-native';
import HomeIcon from './HomeIcon'

class HomeScreenRow extends React.Component {

  render() {
    const { navigate } = this.props.navigation;
    return (
      <View style={styles.container} >
          <TouchableOpacity
              style={styles.iconContainer}
              onPress={() => navigate(this.props.left.route)}>
              <HomeIcon 
                  label={this.props.left.title} 
                  width={120}
                  height={100}
                  colorTop="#1C4E84"
                  colorBottom="#0F2843"
                  iconSource={this.props.left.src} 
                  iconWidth={30}
                  iconHeight={30}
                  />
          </TouchableOpacity>

          <TouchableOpacity
              style={styles.iconContainer}
              onPress={() => navigate(this.props.right.route)}>
              <HomeIcon 
                  label={this.props.right.title} 
                  width={120}
                  height={100}
                  colorTop="#1C4E84"
                  colorBottom="#0F2843"
                  iconSource={this.props.right.src} 
                  iconWidth={30}
                  iconHeight={30}
                  />
          </TouchableOpacity>
      </View>
    )
  }
}

const styles = StyleSheet.create({
    container: {
        flex: 1,
        padding: 10,
        flexDirection: 'row',
        alignItems: 'center',
    },
});

export default HomeScreenRow;

Two entry points

React Native has separate entry points for Android and iOS apps. This allows you to organize your app according to the platform.

IDE?

Mostly use text editors that you use to edit Javascript. React generates Xcode and Android projects. You can edit those if you'd like to add platform specific resources (like iOS @2x, @3x assets)
